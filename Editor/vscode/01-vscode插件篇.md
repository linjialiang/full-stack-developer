# vscode 插件篇

vscode 插件非常丰富，大类分为：内置扩展和社区插件。

## 内置扩展

相较于 Atom 来讲，vscode 的内置扩展更加适用于实际编码，对于前端开发人员甚至可以不需要安装社区扩展。内置扩展如下：

| 内置扩展分类                           |
| -------------------------------------- |
| [语法基本扩展](./core/语法基本扩展.md) |
| [功能性扩展](./core/功能性扩展.md)     |
| [主题扩展](./core/主题扩展.md)         |

## 社区插件

内置扩展是集成在 vscode 内部的，我们无法看到具体的源码，但是社区插件是以源码的形式出现的，我们甚至可以直接修改它

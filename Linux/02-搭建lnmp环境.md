# 搭建 LNMP 环境

Linux 下推荐大家都使用 LNMP 作为开发环境，毕竟部署环境 LNMP 才是最佳选择。

## httpd 与 nginx 比较

| 对比项目      | httpd      | nginx             |
| ------------- | ---------- | ----------------- |
| 对于初学者    | 掌握容易   | 掌握困难          |
| 资源占用      | 普通       | 极低              |
| 抗高并发      | 较差       | 极强              |
| 稳定性        | 超稳       | 一般              |
| 处理请求      | 同步多进程 | 异步多线程        |
| 处理 php 脚本 | 模块处理   | 转发 php-fpm 处理 |

## LNMP 概念

本教程所说的 LNMP 指的是： `Debian + Nginx + MariaDB + PHP`

# 程序员概念大全

1. 挂起

   | 概念 | `挂起` 是指在操作系统行程管理将前台的行程暂停并转入后台的动作。                |
   | ---- | ------------------------------------------------------------------------------ |
   | 特点 | 将进程挂起可以让用户在前台执行其他的行程；                                     |
   | 优点 | 将进程挂起通常可以释放已经占有的系统资源，如内存等（CPU 除外）；               |
   | 操作 | 需要时，用户可以通过指令将被挂起的进程从后台转入前台，并从暂停处开始继续运行。 |

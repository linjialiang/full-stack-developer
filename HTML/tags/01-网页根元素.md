# 网页根元素

| 元素 | 描述                                                                                                     |
| ---- | -------------------------------------------------------------------------------------------------------- |
| html | 网页的顶层容器，也称为根元素（root element），其他元素都是它的子元素。一个网页只能有一个 `<html>` 标签。 |

该标签的 lang 属性，表示网页内容默认的语言:

| lang 属性值  | 描述     |
| ------------ | -------- |
| lang=en      | 英语     |
| lang=zh      | 中文     |
| lang=zh-Hans | 简体中文 |
| lang=zh-Hant | 繁体中文 |

通常国人的网页应该这样标记：

```html
<html lang="zh-Hans"></html>
```
